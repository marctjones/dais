# Dais Rust TUI Shortcuts

## Global

| Key | Action |
|-----|--------|
| `q` | Quit |
| `tab` / `right` | Next tab |
| `shift+tab` / `left` | Previous tab |
| `j` / `down` | Move down |
| `k` / `up` | Move up |
| `r` | Refresh current tab |
| `c` | Compose a private post |
| `b` | Compose a public post |
| `e` | Compose an encrypted post |
| `/` | Search |
| `a` | Approve selected follower |
| `x` | Reject selected follower |
| `m` | Mark notification read |
| `u` | Unblock selected actor |

## Compose

| Key | Action |
|-----|--------|
| `ctrl+s` | Send |
| `tab` | Switch field |
| `v` | Toggle audience |
| `p` | Toggle protocol |
| `e` | Toggle E2EE |
| `esc` | Cancel |

Private posts are the default. Public and encrypted compose modes are one key away.
