# Dais Desk Design Alignment Matrix (v0.39–v0.42)

Updated: 2026-06-20

Goal: verify the greenfield Slint client behavior matches the design alignment
objectives without inheriting legacy-tainted workflows.

## Milestone 0.39

### Scope
Finalize interaction and keyboard contract across list, inspector, and actions.

- [x] Top-level modes are fixed to **Home / People / Server**.
- [x] Home list is attention-first (`notifications`, `DMs`, then content).
- [x] List/inspector split is present and contextual actions are in the inspector when
  secondary list actions are suppressed.
- [x] Action set is bounded to safe, discoverable verbs (`Reply`, `Approve`, `Block`, `Inspect`,
  `Follow`, etc.).

Evidence:
- `apps/dais-desk/src/lib.rs`
- `apps/dais-desk/src/lib.rs` tests:
  - `home_today_rows_are_attention_first`
  - `inspector_exposes_secondary_actions_when_list_suppresses_them`

## Milestone 0.40

### Scope
Refine core IA and calmer primary screens.

- [x] Navigation is constrained to **Home/People/Server** as only top-level mode tabs.
- [x] People mode separates discovery from relationship screens (`Find`, `Relationship`,
  `Friends`, `Followers`, `Following`).
- [x] Home lanes prioritize today/feed and notification attention before long-form content.
- [x] Home and People relationships keep secondary actions minimal by default.

Evidence:
- `apps/dais-desk/src/lib.rs`
- `mode_nav` and `screen_nav` implementations
- `fixture_projection_has_all_primary_modes`
- `fixture_people_mode_has_expected_screen_order`

## Milestone 0.41

### Scope
Visual semantics and noise reduction.

- [x] Remove raw protocol disclosure from social row subtitles.
- [x] Notifications and moderation replies use readable titles and cleaned previews.
- [x] External URLs are extracted even in HTML snippets so link actions remain discoverable.
- [x] HTML/script/style content is stripped from rendered previews.

Evidence:
- `apps/dais-desk/src/lib.rs`:
  - `post_row`
  - `notification_row`
  - `moderation_reply_row`
  - `clean_text`
  - `preview_markdown_safe`
  - `extract_first_url`
- UI tests:
  - `notification_rows_expose_contextual_actions`
  - `notification_row_provides_readable_preview`
  - `notification_row_from_html_exposes_link_action`
  - `extract_first_url_handles_html_and_markdown`
  - `moderation_reply_row_humanizes_status_and_flags`
